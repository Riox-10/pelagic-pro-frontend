# PELAGIC PRO S.A.S Website

Site web professionnel pour PELAGIC PRO S.A.S développé avec :

- Next.js App Router
- TypeScript
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
```

Puis ouvre :

```bash
http://localhost:3000
```

## Images

Les images sont déjà placées comme placeholders dans `public/`.
Remplace-les par les vraies images en gardant exactement les mêmes noms.

```txt
public/
  images/
    logo.png

  company/
    company-1-production-line.jpg
    company-2-processing-room.jpg
    company-3-machinery.jpg
    company-4-packaging-line.jpg

  certificates/
```
